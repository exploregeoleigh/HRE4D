A quick program to calculate the phase difference of two files
Call with [program] [file1] [file2] [1 for phase, 0 for res]
Input: two sigma files, one high, one low
Output: a sigma file named from both files, with some math performed on them. Check the program source for more info

You may need to edit the values for the sstarting model.

If you dont know why this is here, check out the document on running VOI's with E4D, probably in this directory
